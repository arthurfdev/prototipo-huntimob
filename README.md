# HuntImob Frontend

Sistema de gestão imobiliária desenvolvido com Nuxt 3.

## Tecnologias

- **Nuxt 3** - Framework Vue.js
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Framework CSS utilitário
- **Pinia** - Gerenciamento de estado
- **Element Plus** - Componentes UI
- **PWA** - Progressive Web App

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
